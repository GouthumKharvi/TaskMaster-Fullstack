# TaskMaster – Full Stack Task Board

TaskMaster is a clean, full-stack task management application built with FastAPI and React.

It demonstrates rapid full-stack development, clean API design, and a polished, user-friendly UI within a tight time constraint.

## Features
- Add, delete, and complete tasks
- Visual progress indicator
- Clean modern UI with Tailwind CSS
- FastAPI backend with in-memory storage
- Frontend–backend integration via REST APIs

## Tech Stack
- Backend: Python, FastAPI
- Frontend: React, Vite, Tailwind CSS
- Styling: Tailwind CSS
- Storage: In-memory (runtime only)

## Purpose
Built as a Vibe Coding Assignment to demonstrate speed, clarity, and good engineering judgment.
